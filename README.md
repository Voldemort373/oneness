# Oneness

Oneness is a decentralised, censor-free, open-source social media platform, built on Arweave (with Bundlr) and Polygon.

## Theme colors
- **Blue**: `#06bcc1`
- **Green**: `#66c184`
- **Orange**: `#f4b860`
- **Red**: `#ff5a5f`
